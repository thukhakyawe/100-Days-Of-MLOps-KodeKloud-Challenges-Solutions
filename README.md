# 100 Days of MLOps

A hands-on 100-day journey to master MLOps engineering through practical projects, reproducible pipelines, model deployment, monitoring, CI/CD, Kubernetes orchestration, and production-grade ML systems.

## 🚀 What You'll Learn

This repository covers the complete MLOps lifecycle:

* Python environments and ML project structuring
* DVC for dataset and experiment versioning
* MLflow for experiment tracking and model registry
* Training pipelines with Scikit-Learn and PyTorch
* Docker containerization for ML workloads
* Model serving with Flask, FastAPI, and BentoML
* Monitoring with Prometheus, Grafana, and Evidently
* CI/CD pipelines for ML systems
* Kubernetes, Argo Workflows, Kubeflow, and GitOps
* Production-grade deployment and automated retraining

## 📚 Structure

The roadmap is divided into progressive phases:

1. Foundations & Project Setup
2. Data Versioning with DVC
3. Experiment Tracking with MLflow
4. Model Training & Optimization
5. Feature Stores & Data Quality
6. Docker & Containerization
7. Model Serving & Deployment
8. Monitoring & Observability
9. CI/CD for Machine Learning
10. Kubernetes & Workflow Orchestration
11. End-to-End Production Capstone Projects

## 🎯 Goal

Build real-world MLOps skills by implementing production-ready ML systems step by step over 100 days.

## 🛠️ Tech Stack

Python • DVC • MLflow • Docker • FastAPI • Flask • PyTorch • Scikit-Learn • Optuna • Evidently • Prometheus • Grafana • Kubernetes • Argo Workflows • Kubeflow • GitHub Actions

By the end of this roadmap, you'll be able to design, automate, deploy, monitor, and scale end-to-end machine learning systems in production environments.
